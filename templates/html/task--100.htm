Привет!

Ниже представлена ознокомительная информация с языком разметки MessageUP.

Поддержка html

MUP можно использовать совместно с html кодом

Правила использования

Все @ операторы должны быть отделены пустой строкой сверху и *снизу
*операторы которые не модифицируют текст

1. Стандартный блок сообщения

@msg
Text

2. Предупреждающий блок сообщения

@msg-warn
Text

3. Заголовок в виде блока сообщения

@msg-header

Text

4. Перенос строки

@n
Text

Text

5. Заголовок

@header

Text

6. Кнопка-ссылка

@btn-link
Text!Test
Link!https://github.com/egorgasay
Test

7. Жирный текст

@text

Text


8. Обычный текст (по умолчанию)

@st
Text
